# Semantic Timeline Panel (Frontend Example) This is a standalone, accessible, and responsive HTML/CSS/JS panel for visualização da Semantic Timeline, pronto para consumir o endpoint `/v1/passive/regions/{region_id}/semantic-timeline`. ## Como usar - Substitua `REGION_ID` pelo id da região desejada. - Abra o arquivo em um navegador moderno. --- Semantic Timeline Panel
Semantic Timeline
--- ## Documentação do endpoint ### GET /v1/passive/regions/{region_id}/semantic-timeline - **Descrição:** Retorna a timeline semântica de eventos para uma região. - **Parâmetros:** - `region_id` (path): ID da região - `limit` (query, opcional): número máximo de entradas - **Resposta:** - `scope`, `scope_id`, `dominant_status`, `entries[]` (array de eventos) - Cada entrada: `event_type`, `site_name`, `status`, `support_count`, `risk_score`, `risk_delta` (com `classification`, `explanation`), `status_summary`, `first_observed_at_unix_seconds`, `last_observed_at_unix_seconds` - **Exemplo de uso:** - `GET /v1/passive/regions/iberian-peninsula/semantic-timeline?limit=10` --- ## Feedback de usuários - Adicione botões de feedback ou links para coleta de sugestões. - Exemplo: `Enviar feedback` - Integre analytics para monitorar uso e pontos de abandono. --- Este painel pode ser expandido conforme feedback dos usuários e necessidades de UX.
